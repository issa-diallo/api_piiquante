# App API Piiquante

Piiquante is dedicated to the creation of **spicy sauces**

Frontend is available via:
<https://github.com/issa-diallo/api_piiquante.git>

## Install

```sh
npm install
```

## :tada: Run

```sh
npm run dev
```

## :rotating_light: Linting

```sh
npm run lint
npm run format
```
